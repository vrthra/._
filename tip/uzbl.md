### Uzbl

These are my bindings from this repo.

#### Tabbed

- New `^t`
- Go : `:o <uri>`
- Edit Url : `O <uri>`
- with clipboard : `gY`
- Close : `gC`
- First,Last : g< g>
- Next : gt gT gi<index>

#### Window

- clone :clone

#### Sessions

- save: gs <file>
- load: glo <file>
- delete: gd <file>
- list: gli

### Load URL

- from history: U
- from bookmarks: u

### Book marks

- ^s
- ^l
